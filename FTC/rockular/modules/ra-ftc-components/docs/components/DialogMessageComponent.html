DialogMessageComponent provides visual representation for dialog message.

Message dialog

This module provides methods and definition to open message dialog with Angular material2 dialog, MatDialog (https://material.angular.io/components/dialog/overview).

Configuration

Dialog have to have specific configuration passed in to open with dialog message component DialogMessageComponent.

It is possible to set these configuration properties:

  • title - title for message dialog
  • message - main message text to display in message dialog
  • optionalMessage - as name implies this is optional message text with extra top margin in dialog, it is positioned under main message text
  • messageType - type of dialog to be shown, it changes dialog visual
  • buttons - buttons configuration for dialog footer

See examples.

const config: IDialogMessageConfig = {
    title: 'Message dialog title',
    message: 'Main message to appear in dialog',
    messageType: 'warning', // warning | info | error
    buttons: [
        {
            label: 'Yes'
            action: (event, dialogRef: MatDialogRef<DialogMessageComponent>) => {
                dialogRef.close();
            }
        }
    ],
    optionalMessage: ''
};

How to open message dialog

...

constructor(
    private messageDialogSvc: MessageDialogService
    ) {}

openMessageDialog() {
    const config: IDialogMessageConfig = {
        title: 'Message dialog title',
        message: 'Main message to appear in dialog',
        messageType: 'info',
        buttons: []
    };
    messageDialogSvc.openMessageDialog(config);
}

constructor(

    ) {}

openMessageDialog() {
    const buttons = [{
        label: 'Close',
        action: (event, dialogReference) => {
            dialogReference.close();
        }
    }];
    messageDialogSvc.openWarningDialog(
        'Message dialog title',
        'Main message to appear in dialog',
        buttons,
        'Optional message');
}
