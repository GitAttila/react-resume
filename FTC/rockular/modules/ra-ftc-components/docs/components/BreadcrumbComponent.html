BreadcrumbComponent this component generates a breadcrumb navigation from specified items.

Breadcrumb

This module provides methods and definition to create a breadcrumb component.

Configuration

Component can be configured by passing breadcrumb attribute which is an array of type IBreadcrumbItem.

<ra-ftc-breadcrumb [breadcrumbs]="items" (itemClicked)="onItemClick($event)"></ra-ftc-breadcrumb>

It is possible to set these component properties:

  • breadcrumbs - The array of IBreadcrumbItem to display.
  • breadcrumbs.fqn - Item fqn.
  • breadcrumbs.name - Item name.
  • itemClicked - Callback function to be called after click on a breadcrumb item. Function expects a parameter of type IBreadcrumbItem which is the clicked item.

See examples.

const breadcrumbs: IBreadcrumbItem[] = [
          {
            fqn: 'MyEnterprise',
            name: 'MyEnterprise'
          },
          {
            fqn: 'MyEnterprise.Rockwell',
            name: 'Rockwell'
          },
          {
            fqn: 'MyEnterprise.Rockwell.Prague',
            name: 'Prague'
          },
          {
            fqn: 'MyEnterprise.Rockwell.Prague.Gateway_CLX',
            name: 'Gateway CLX'
          },
          {
            fqn: 'MyEnterprise.Rockwell.Prague.Gateway_CLX.Long_name_of_fqn_item',
            name: 'Long name of fqn item'
          }
        ];
