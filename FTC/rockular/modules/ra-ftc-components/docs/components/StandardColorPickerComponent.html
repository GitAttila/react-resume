StandardColorPickerComponent provides visual representation for standard color picker.

Standard color picker

This module provides methods and definitions to show selected color and also allows user to pick a color from component colorpicker.

Configuration

Input bindings

It is possible to set these configuration properties (as attribute property binding):

  • appearance - optional settings to allow set visualization as circle or square.
    const appearance: ColorPickerAppearance = 'square';  // default is 'circle'
  • presetColors - array of color definitions.
    const presetColors: ColorPickerInput[] = [
  { h: 10, s: 50, l: 90, a: 0.8 },
  { r: 80, g: 190, b: 255, a: 0.5 },
  '#EEECE1',
];
  • value - actual color value
    // value has no default value
// value of type <ColorPickerInput> allows: string | IColorPickerRGB | IColorPickerRGBA | IColorPickerHSLA IColorPickerHSLA | TinyColor
// value has no default value
// value of type <ColorPickerInput>
const colorValue: string = '#C19859'; // also 'rgb(10, 25, 30)', 'rgba(10, 25, 30, 1)' is supported
const colorValue: IColorPickerHSL = {h: 10, s: 50, l: 90};
const colorValue: IColorPickerHSLA = {h: 10, s: 50, l: 90, a: 0.8};
const colorValue: IColorPickerRGB = {r: 80, g: 190, b: 255};
const colorValue: IColorPickerRGBA = {r: 80, g: 190, b: 255, a: 0.5};
const colorValue: IColorPickerHSV = {h: 10, s: 50, v: 90};
const colorValue: IColorPickerHSVA = {h: 10, s: 50, v: 90, a: 0.5};
const colorValue: TinyColor = new TinyColor('rgb(10, 25, 30)');
  • disabled - flag indicating disabled state of component, user is not able to select color
    const disabled: boolean = false; // default as false
  • showDropdown - flag indicating to show dropdown
    const showDropdown: boolean = false; // default as false
  • disableAlpha - flag indicating to hide alpha channel selection from picker
    const disableAlpha: boolean = false; // default as false

Output bindings

  • colorSelected - triggered when color is selected from color picker, this is of type ColorPickerColor
    onColorSelected(output) {
  // output ~
  output = {
      hex: '#c0504d', 
      rgb: { r: 192, g: 80, b: 77, a: 1 }, 
      rgbString: 'rgb(192, 80, 77)', 
      hsl: { h: 1.5652173913043497, s: 0.47717842323651455, l: 0.5274509803921569, a: 1 }, 
      hslString: 'hsl(2, 48%, 53%)', 
      hsv: { h: 1.5652173913043497, s: 0.5989583333333334, v: 0.7529411764705882, a: 1 }, 
      hsvString: 'hsl(2, 48%, 53%)',
      source: '#c0504d'
  };
}
  • opened - triggered when popup is opened
  • closed - triggered when popup is closed

Usage in template

<ra-ftc-Standard-color-picker [presetColors]="presetColors" [appearance]="appearance" [value]="colorValue" [disabled]="disabled" [showDropdown]="showDropdown" [disableAlpha]="disableAlpha" (colorSelected)="callbackWhenColorSelected($event)"></ra-ftc-Standard-color-picker>
