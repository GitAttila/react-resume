Inputs

value

Value to be rendered under title

Type : string

valueTitle

Title for value

Type : string

<ra-ftc-value-title> provides visual for value and its title. This visual is suitable as content for cards visuals.

Configuration

It is possible to set these properties:

  • valueTitle - title for value
    const valueTitle: string = 'Name';
  • value - value to be shown under title
    const value: string = 'Robert Tester';

Examples

<ra-ftc-value-title [value]="value" [valueTitle]="valueTitle"></ra-ftc-value-title>
