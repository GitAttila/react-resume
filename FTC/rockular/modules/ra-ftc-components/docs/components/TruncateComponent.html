<ra-ftc-truncate> provides a truncate mechanism for text at left, center and right side of given text

Configuration

It is possible to set these properties:

  • text - text to apply truncate to
    const text: string = 'Some text to be truncated.';
  • multiline - flag indicating to truncate in multiline, defaults to false
    const multiline: boolean = false;
  • position - position for truncate to appear at, 'left' | 'center' | 'right', defaults to 'right'
    const position: TruncatePosition = 'right';

Examples

<div fxLayout="row">
    <div fxFlex="0 0 100px" fxLayout="column"><!-- container with width of 100px -->
        <ra-ftc-truncate fxFlex="0 0 180px" [text]="text" [multiline]="multiline" position="left"></ra-ftc-truncate> <!-- truncate with height of 180px -->
    </div>
</div>
