IconPickerComponent this component allows to pick an icon from configurable set of icons.

Icon picker

This module provides methods and definition to show selected icon and also allows user to pick an icon from configurable set of icons.

Howto configure module to load initial set of icons

You have to import RaFtcIconPickerModule.forRoot() in the root NgModule of your application. This method allows you to configure the RaFtcIconPickerModule by specifying an array of fonts with path to json file containing icons that will be loaded to icon picker dropdown by default.

The forRoot static method is a convention that provides and configures services at the same time. Make sure you only call this method in the root module of your application, most of the time called AppModule.

Module configuration properties:

  • fonts - array of fonts to be displayed inside icon picker dropdown.
  • fonts.name - font name represented by tab title inside icon picker dropdown.
  • fonts.iconsPath - path to json file containing font icons to be loaded into icon picker dropdown. You can copy Material icons json and RaIcons json or provide a custom json file.
  • fonts.fontSet - font set that the icons are part of.
  • fonts.useLigature - whether to show icons by using ligatures or whether to display icons by defining a CSS class for each icon glyph, which typically uses a :before selector to cause the icon to appear. Read more about ligatures in: https://material.angular.io/components/icon/overview
@NgModule({
    imports: [
        BrowserModule,
        HttpClientModule,
        RaFtcIconPickerModule.forRoot({
          fonts: [
            {
              name: 'Material',
              iconsPath: './assets/fonts/material-icons/material-icons.json',
              fontSet: 'material-icons',
              useLigature: true
            },
            {
              name: 'Ra Icons',
              iconsPath: './assets/fonts/ra-icons/ra-icons.json',
              fontSet: 'ra-icons',
              useLigature: false
            }
          ]
        })
    ],
    bootstrap: [AppComponent]
})
export class AppModule { }

Configuration

Input bindings

It is possible to set these configuration properties (as attribute property binding):

  • fontList - array of fonts to display inside icon picker dropdown
    const fontList: IFont[] = [
{
  "name": "Material",
  "fontSet": "material-icons",
  "useLigature": true,
  "icons": [
    {
      "name": "3d_rotation",
      "code": "3d_rotation"
    },
    {
      "name": "ac_unit",
      "code": "ac_unit"
    }
  ]
},
{
  "name": "Ra Icons",
  "fontSet": "ra-icons",
  "useLigature": false,
  "icons": [
    {
      "name": "ra-icon-home",
      "code": "ra-icon-home"
    },
    {
      "name": "ra-icon-settings",
      "code": "ra-icon-settings"
    },
    {
      "name": "ra-icon-settings-filled",
      "code": "ra-icon-settings-filled"
    }
  ]
}
];
  • selectedIcon - selected icon
    const selectedIcon: ISelectedIcon = {
name: 'ra-icon-settings',
code: 'ra-icon-settings',
fontSet: 'ra-icons',
useLigature: false
};
  • visibleFontSets - array of strings with names of font sets to display inside icon picker dropdown. By default all font sets are visible.
    const visibleFontSets: string[] = [ "material-icons", "ra-icons" ];
  • disabled - flag indicating disabled state of component
    const disabled: boolean = false; // default as false
  • showDropdown - flag indicating to show dropdown
    const showDropdown: boolean = false; // default as false

Output bindings

  • iconSelected - triggered when icon is selected from one of font sets
    onIconSelected(output) {
  // output ~
  output = {
      name: 'ra-icon-settings',
      code: 'ra-icon-settings',
      fontSet: 'ra-icons',
      useLigature: false
  };
}
  • selectionCleared - triggered when selected icon has been cleared
    onSelectionCleared() {
}
  • opened - triggered when popup is opened
  • closed - triggered when popup is closed

Examples

<ra-ftc-icon-picker [fontList]="fontList" [visibleFontSets]="visibleFontSets" [disabled]="disabled" [showDropdown]="showDropdown" (iconSelected)="selectedIcon = $event" (selectionCleared)="selectedIcon = undefined"></ra-ftc-icon-picker>
