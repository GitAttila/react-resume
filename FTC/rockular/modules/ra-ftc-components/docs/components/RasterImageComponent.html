<ra-ftc-raster-image> provides a raster image display with options.

Configuration

Raster Image component takes 6 parameters

<ra-ftc-raster-image 
    [image]="image" <!-- URL of the image -->
    [fitToSize]="fitToSize" <!-- boolean, if true resizes the image -->
    [keepRatio]="keepRatio" <!-- if true and if fitToSize is true the image will resize without keeping the original image's aspect ratio -->
    [horizontalAlign]="horizontalAlign" <!-- 'Center' | 'Left' | 'Right'. Will fall back to 'Center' if the value is invalid -->
    [verticalAlign]="verticalAlign" <!-- 'Center' | 'Top' | 'Bottom'. Will fall back to 'Center' if the value is invalid -->
    [backgroundColor]="backgroundColor"> <!-- string in '#<hex value>' format. -->
</ra-ftc-raster-image>
