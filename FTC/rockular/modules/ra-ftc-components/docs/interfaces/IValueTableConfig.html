<ra-ftc-value-table> provides an expandable value table with categories.

value-table content

Content can be added by passing a variable in item attribute of type IValueTableItems (which contains an attribute of type IValueTableListItem)

Configuration

Component can be configured by passing variable in config attribute of type IValueTableConfig.

<ra-ftc-value-table [config]="config" [items]="items"></ra-ftc-value-table>
const config: IValueTableConfig = {
    allowMultipleExpansions: true, // enables the option to expand all table categories
    displayMode: 'default', // 'default' | 'flat'. Changes style of the table
    expanded: true, // expands/collapses all categories
    disableCollapse: false // expands all categories, hides the collapse/expand button and disables collapsion
};

Examples

Simple example with a single category

...
public valueTableConfig: IValueTableConfig = {
    allowMultipleExpansions: true,
    displayMode: 'default',
    expanded: true,
    disableCollapse: false
};

public valueTableItems: IValueTableItems[] = [
    {
        name: 'test',
        value: [
            {
                name: 'test1_1',
                value: 3.14159,
                engineeringUnit: 'm/s', // will only apply if value is a number
                precision: 2 // will only apply if value is a number
            },
            {
                name: 'test1_2',
                value: 'test value 2'
                isBadQuality: true // shows an 'unknown' icon from ra-ip-icons next to the value
            }
        ]
    }
];
