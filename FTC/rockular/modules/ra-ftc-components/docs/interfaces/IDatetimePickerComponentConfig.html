DatetimePickerComponent component for selecting date and time.

Datetime Picker

This module provides methods and definition for creating Datetime Picker component which was based on mat-datetimepicker (https://github.com/kuhnroyal/mat-datetimepicker).

Datetime Picker Design

User can choose a date.

After choosing date, the user can choose time.

The user is choosing minutes.

Configuration

Component can be configured by optional attributes 'value' and 'config'. Without any of these attributes, input field stays empty until user chooses date and time.

<ra-ftc-date-time-picker [value]=datetimeValue [config]=config></ra-ftc-date-time-picker>

Properties:

  • value - Value represents datetime value, which can be a type of string or a type of moment object. (https://momentjs.com/docs/)
  • config - Value of config represents rules for parsing and display settings of component.

See examples:

<ra-ftc-date-time-picker [value]=momentObject [config]="{type: "time", format: "LT"}"></ra-ftc-date-time-picker>
  • value - object created by momentJS
  • config - config sets rules on displaying -> show only time, do not show date
<ra-ftc-date-time-picker [value]="08/05/2017 12:11 AM" [config]="{type: "datetime", format: "l LT", popUpHeaderDateLabelFormat: "ddd, DD MMMM"}"></ra-ftc-date-time-picker>
  • value - datetime string
  • config - config sets rules on parsing input string -> shows date without zeros before single digits and full name of months in pop up header

Implicit value of config.

Properties type and format are required to be filled. Property popUpHeaderDateLabelFormat is optional.

{
  type: 'datetime',
  format: 'L LT',
  popUpHeaderDateLabelFormat?: 'ddd, DD MMMM'
};

For information about format settings (L, LL, MMMM, etc) see https://momentjs.com/docs/
