MapComponent component for displaying an interactive map.

Map

This module provides methods and definition to create a map with Leaflet packages for Angular.io (v2+), Leaflet map (https://leafletjs.com/).

Configuration

Component can be configured by passing config attribute of type IMapConfig.

<ra-ftc-map [config]="config"></ra-ftc-map>

It is possible to set these configuration properties:

  • center - The geographic coordinates of center of map. Default value is '0, 0'.
  • zoom - The zoom level of map, minimum value is 0, maximum value is 18
  • hideLegend - Disable legend in Map widget. Default value is false.
  • clusterSize - Radius of cluster for markers, minimum vlaue is 10, maximum is 20. Default value is 15.
  • clusterPopupMessage - Overrides message that appears above cluster markers.
  • hideClusterPopup - Disable cluster popup in Map widget. Default value is false.
  • popupVerbosity - The MapVerbosity enum with options which minimum info display in cluster popup. Values are - disabled, low, medium, high, urgent.
  • indication - An object with properties to customize legend for Map widget.
  • indication.defaultIndication - Object of type IMapBasicIndication to override default legend member.
  • indication.defaultIndication.color - The color of default legend member. Default value is hexadecimal string '#7f7f7f'.
  • indication.badQualityIndication - Object of type IMapBasicIndication to override bad quality legend member.
  • indication.badQualityIndication.color - The color of bad quality legend member. Default value is hexadecimal string '#000000'.
  • indication.customIndication - The array with custom members of legend. Array items are of type IMapIndication.
  • indication.customIndication.from - The beginning of indication range.
  • indication.customIndication.to - The end of indication range.
  • indication.customIndication.color - The color of legend member as hexadecimal string.
  • indication.customIndication.label - The text of legend member.
  • indication.customIndication.verbosity - The MapVerbosity enum with option which verbosity level range is. Values are - disabled, low, medium, high, urgent.
  • providers - The array of IMapProvider object with specification of custom map layers.
  • providers.link - Link to the map provider. Default value is 'https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png'.
  • providers.name - Name of the map provider that will be displayed in map layer panel. Default value is 'Open Street Map'.

See examples.

const config: IMapConfig = {
            zoom: 2,
            center: '10, 10',
            hideLegend: true,
            clusterSize: 20,
            clusterPopupMessage: 'Nothing found',
            hideClusterPopup: true,
            popupVerbosity: MapVerbosity.medium,
            zoomPosition: MapControlPosition.bottomright,
            legendPosition: MapLegendPosition.top,
            layersPosition: MapControlPosition.bottomleft,
            providers: [{
              link: 'https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png',
              name: 'OSM'
            }],
            indication: {
              defaultIndication: {
                color: '#7f7f7f'
              },
              badQualityIndication: {
                color: '#000000'
              },
              customIndication: [
                {
                  from: 1,
                  to: 1,
                  color: '#22b14c',
                  label: 'Good',
                  verbosity: MapVerbosity.low
                }, {
                  from: 2,
                  to: 2,
                  color: '#26a9e0',
                  label: 'Maintenance',
                  verbosity: MapVerbosity.medium
                }, {
                  from: 3,
                  to: 3,
                  color: '#f5e11b',
                  label: 'Warning',
                  verbosity: MapVerbosity.high
                }, {
                  from: 4,
                  to: 4,
                  color: '#e22028',
                  label: 'Error',
                  verbosity: MapVerbosity.urgent
                }
              ]
            }
         };

Map markers can be added by passing value attribute which is an array of type IMapMarker.

<ra-ftc-map [value]="markers"></ra-ftc-map>

IMapMarker parameters

  • value - Main property, geographic coordinates of marker.
  • caption - The caption of markers popup.
  • text - The text inside of markers popup.
  • colorIndication - The value which determines color of marker based on legend.
  • colorIndication - The value which determines color of marker based on legend.
  • colorIndicationBadQuality - The value which determines whether the marker has legend bad quality color.
  • drillThrough - The array of IMapDrillThrough objects to add links to markers popup.
  • drillThrough.link - External url link.
  • drillThrough.caption - Caption link.

See examples.

const markers: IMapMarker[] = [
      {
        value: '90, 90',
        caption: 'Marker 2'
      }, {
        value: '50, 14',
        colorIndicationBadQuality: true,
        caption: 'Marker 1',
        text: 'Text',
        drillThrough: [{
          link: 'https://google.com',
          caption: 'Link'
        }],
        colorIndication: 1
      }, {
        value: '20, 10',
        text: 'Text'
      }, {
        value: '20, 10.5'
      }
    ];
