iconSvc
service in module RA.IPCommon.Services

Description

Service for Icon Related Manipulations

Methods

  • getServiceUrl([imageFqn], [size], [effect])

    Constructs a URL to service using given parameters, encoding each parameter with encodeURIComponent() function.

    Parameters
    ParamTypeDetails
    [imageFqn]
    (optional)
    		string

    the FQN of image

    [size]
    (optional)
    		string

    the desired image size; if this parameter is omitted and the effect parameter is passed, the 'small' value for size is used by default

    [effect]
    (optional)
    		string

    the effect name to apply to an image

    Returns
    string

    The URL to IconService with properly encoded parameters, or just the base URL if called with no parameters.
