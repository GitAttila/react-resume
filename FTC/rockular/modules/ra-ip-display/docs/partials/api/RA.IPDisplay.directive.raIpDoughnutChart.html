raIpDoughnutChart
directive in module RA.IPDisplay

Description

This raIpDoughnutChart component is responsible for rendering Doughnut Chart Widget. There is a specific service to handle widget constants, schema and examples, see RA.IPDisplay.service:raIpDoughnutChartConstSvc.

Usage

as element:
<ra-ip-doughnut-chart
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       rb-design-mode="{boolean}">
</ra-ip-doughnut-chart>

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of Doughnut Chart Widget.

Common Parameters

Property Type Details Default value
autoReload Boolean Should the values be auto reload False
autoReloadInterval (optional) Number Time interval to auto reload after 5 sec
lookBackwardHours Number Number of Seconds the value are being displayed for. Special treatment for value 0 (see startTime and endTime parameters) and -1 (see lastSeconds parameter). 600 sec (5 minutes)
lastSeconds (optional) Number Mandatory and used in case when lookBackwardHours equals -1 (Last N seconds). Number of Seconds the value are being displayed for when Last N seconds is chosen.
startTime (optional) Date Start time date. Mandatory and used in case when lookBackwardHours equals 0 (Custom range).
endTime (optional) Date End time date. Mandatory and used in case when lookBackwardHours equals 0 (Custom range).
hideTimeControl (optional) Boolean Should widget time control be hidden False
ignoreDataLimit (optional) Boolean Allow user request raw data results False

Schema Type: String

Property Type Details Default value
value String FQN of the string item in data model undefined
keywordRange (optional) Array The array with string ranges for widget
keywordRange.keywords (optional) String Keywords for range, separated by comma
keywordRange.color (optional) Object The color of box when value is one of keywords. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #22b14c
keywordRange.label (optional) String The label for range color. This label is present in chart's legend ''
dataValues String Switch visualization between only doughnut, douhnut with callouts (percentage or value), doughnut with table (percentage, value, value and percentage "none" , "durationTable", "percentageTable", "durationCallouts", "percentageCallouts", "percentageDurationTable"
hideLegend (optional) Boolean Should chart legend be hidden False
hideTooltip (optional) Boolean Should chart tooltip be hidden False

Schema Type: Number

Property Type Details Default value
value String FQN of the numeric item in data model undefined
ranges (optional) Array The array with ranges for widget
ranges.from (optional) Number The begining of range
ranges.to (optional) Number The end of range
ranges.color (optional) Object The color for chart bar when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #22b14c
ranges.label (optional) String The label for range color. This label is present in chart's legend ''
dataValues String Switch visualization between only doughnut, douhnut with callouts (percentage or value), doughnut with table (percentage, value, value and percentage "none" , "durationTable", "percentageTable", "durationCallouts", "percentageCallouts", "percentageDurationTable"
hideLegend (optional) Boolean Should chart legend be hidden False
hideTooltip (optional) Boolean Should chart tooltip be hidden False
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget should be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbDesignModeboolean

Flag which indicates whether the widget is in design mode.
