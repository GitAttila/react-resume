raIpLayoutPlanner
directive in module RA.IPDisplay

Description

The raIpLayoutPlanner directive is responsible for layout container size and counts with ratio, width and height. This directive does not create new scope.

Usage

as element:
<ra-ip-layout-planner
       ratio="{Number}"
       min-width="{Number}"
       min-height="{Number}">
</ra-ip-layout-planner>

Parameters

ParamTypeDetails
ratioNumber

Attribute parameter specifying container ideal ratio.

min-widthNumber

Attribute parameter specifying minimum container width. When minimum height is specified, minimum width attribute is ignored.

min-heightNumber

Attribute parameter specifying minimum container height. When minimum height is specified, minimum width attribute is ignored.
