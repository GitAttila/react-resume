raIpIndicator
directive in module RA.IPDisplay

Description

The raIpIndicator directive is responsible for rendering the Indicator widget.

Usage

as element:
<ra-ip-indicator
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]>
</ra-ip-indicator>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of Indicator. This object contains array of objects (rbParameters.items) with these properties.

Schema type: Boolean

Property Type Details Default value
value Number The main property, Boolean value of indicator
caption (optional) String The text which is displayed as caption of indicator This property is autobound to model .Name property of tag or it is empty
trueColor (optional) Object Color for false values. Color object(see * colorTypeSvc) or hexadecimal String for compatibily reasons #22b14c
falseColor (optional) Object Color for true values. Color object(see * colorTypeSvc) or hexadecimal String for compatibily reasons #e22028
dynamicSwitch (optional) Boolean Enables additonal blink options for indicator. false
animate (optional) String Available only with dynamicSwitch set to true. String Enum with options when switch colors (When true, When false) When true
flash (optional) String Available only with dynamicSwitch set to true. String Enum with options how fast switch colors (Slow, Normal, Fast) Normal
flashColor (optional) Object Available only with dynamicSwitch set to true. Color object(see * colorTypeSvc) or hexadecimal String for compatibily reasons #7f7f7f

Schema type: Number

Property Type Details Default value
value Number The main property, Number value of indicator
caption (optional) String The text which is displayed as caption of indicator This property is autobound to model .Name property of tag or it is empty
numberDynamicSwitch (optional) Boolean Enables additonal blink options for outOfRangeColor. false
flash (optional) String Available only with numberDynamicSwitch set to true. String Enum with options how fast switch colors (Slow, Normal, Fast) Normal
flashColor (optional) Object Available only with numberDynamicSwitch set to true. Color object(see * colorTypeSvc) or hexadecimal String for compatibily reasons #7f7f7f
outOfRangeColor (optional) Object The color of indicator when value is not in any Range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #e22028
ranges (optional) Array Number schema only. The array with ranges for indicator
ranges.from (optional) Number Number schema only. The beggining of range
ranges.to (optional) Number Number schema only. The end of range
ranges.color (optional) String Number schema only. The color of indicator when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #7f7f7f
ranges.rangeDynamicSwitch (optional) Boolean Number schema only. Enables additonal blink options for range color. false
ranges.flash (optional) String Available only with rangeDynamicSwitch set to true. String Enum with options how fast switch colors (Slow, Normal, Fast) Normal
ranges.flashColor (optional) Object Available only with rangeDynamicSwitch set to true. Color object(see * colorTypeSvc) or hexadecimal String for compatibily reasons #7f7f7f
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.
