gridstackSvc
service in module RA.IPDisplay

Description

Facade to gridstack API.

Methods

  • createGridstack(staticGrid, useMobileCellHeight, onResizeCallback, onDragStopCallback, DaD)

    Initializes and configures the gridstack object. This gridstack object is created on top of the HTML element with "gridstack-wrapper" id. This ID is then suffixed by the ID of GridstackWrapper object.

    The drag area of the gridstack cells has to have

    Parameters
    ParamTypeDetails
    staticGridBoolean

    Indication whether the gridstack grid should be static (cannot be changed by user interactions).

    useMobileCellHeightBoolean

    Indication whether the height of each gridstack cell should be defined by service itself (mobileCellHeight constant).

    onResizeCallbackFunction

    Callback function which is called when gridstack cell is resized. This parameter is useless when staticGrid parameter is set to true.

    onDragStopCallbackString

    Callback function which is called when gridstack cell is moved. This parameter is useless when staticGrid parameter is set to true.

    DaDString

    Identifier of D&D provider. 2 values are accepted - RA for MFT D&D or kendo for kendo D&D. RA is used by default.

    Returns
    Object

    Instance of GridstackWrapper
