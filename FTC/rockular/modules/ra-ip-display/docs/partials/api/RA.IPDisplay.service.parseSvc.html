parseSvc
service in module RA.IPDisplay

Description

Provides parsing methods e.g. for GPS coordinates.

Methods

  • parseLatLong(actualValue, modifyByModulo)

    Returns latitude and longitude from string input. Supported formats:

    • decimal 41.553291, -81.747669
    • degrees minutes seconds 40° 26′ 46″ N, 79° 58′ 56″ W
    • degrees decimal minutes 40° 26.767′ N, 79° 58.933′ W
    • decimal degrees 40.446° N, 79.982° W
    Parameters
    ParamTypeDetails
    actualValuestring

    string to parse latitude and longitude from

    modifyByModuloBoolean

    modify lat and long by modulo

    Returns
    Array

    array with latitude and longitude [lat, long] or undefined in case input string does not contain valid coordinate data
