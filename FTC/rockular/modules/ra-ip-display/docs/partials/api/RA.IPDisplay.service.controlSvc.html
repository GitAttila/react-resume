Description

Provides functionality which is used across widget controls.

Methods

  • equalValuesWithoutSpecificProperty(newVal, oldVal, propertyName)

    Function to compare objects without specific property specified by propertyName parameter. This function does not modify input parameters directly, it does remove property given by propertyName parameter from copy of newVal and oldVal parameters and then compares equality.

    Parameters
    ParamTypeDetails
    newValObject

    new object value

    oldValObject

    previous object value

    propertyNameString

    name of property without to evaluate

    Returns
    Boolean

    Flag indicating equality of objects without specific property.

  • getRangeColor(value, ranges, defaultRange, min, min, outOfRange)

    Returns the color of the matched color range.

    Parameters
    ParamTypeDetails
    valueNumber

    Current value to which we want to find the matching range.

    rangesArray

    Array of ranges. Each item in array should be object with number properties from and to. Also, in this object should be at least one from following properties:

    • color
    • fontColor
    • flashColor

    These properties should contain color in hexadecimal notation.

    defaultRangeObject

    Object specifing the color which should be used when no range is matched. It should contain at least one from following properties:

    • color
    • fontColor
    • flashColor

    These properties should contain color in hexadecimal notation.

    minNumber

    Minimal value. This is used when outOfRange color is specified.

    minNumber

    Miximal value. This is used when outOfRange color is specified.

    outOfRangeObject

    (Optional) Object specifing the color which should be used when value is less then min or greater then max. If it is not specified, defaultRange is used instead.

    It should contain at least one from following properties:

    • color
    • fontColor
    • flashColor

    These properties should contain color in hexadecimal notation.

    Returns
    Object

    Matched range object.

    Example

    The example below shows usage of getRangeColor method and the visualization of its result.

    • 0-10: Matching 'green' range
    • 11-49: Matching default 'grey' range
    • 50-70: Matching 'yellow' range
    • 71-100: Matching 'orange' range
    • Any other value: Matching outOfRange 'red' range
