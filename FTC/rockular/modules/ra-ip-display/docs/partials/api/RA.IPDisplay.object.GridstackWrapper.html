  • addPlaceholder(lock, event, minimumWidth, minimumHeight)

    Adds placeholder into grid. This placeholder is automatically registered for move events (it presumes that placeholder was added to grid as a reaction on onDragEnter MFT D&D event).

    If placeholder cannot be added to grid, error message is appended to drag helper. This error message has class "dropInvalid" and can be styled by end adopters.

    Note: Use this function only if the event parameter is MFT onDragEnter event.
    Parameters
    ParamTypeDetails
    lockBoolean

    If true all cells (except placeholder) are locked during drag.

    eventObject

    MFT D&D onDragEnter event

    minimumWidthInteger

    optional parameter to allow to check minimum width widget can fill

    minimumHeightInteger

    optional parameter to allow to check minimum height widget can fill

  • adjustHeight()

  • deletePlaceholderIfNeeded(event)

    Deletes the placeholder when dragged outside of gridstack area. If it is not dragged outside, nothing happens.

    Note: Use this function only if the event parameter is MFT onDragLeave event. This function should be refactored in the future since it is created for the purposes of Composer. When used outside of Composer the scope of dragged helper has to have curItem object property. This property has to have dragId property set to "emptyWidget".

    In other words the line of code: 
            event.draggable.element.scope().curItem.dragId
    has to return "emptyWidget" string;
    Parameters
    ParamTypeDetails
    eventObject

    MFT onDragLeave event
