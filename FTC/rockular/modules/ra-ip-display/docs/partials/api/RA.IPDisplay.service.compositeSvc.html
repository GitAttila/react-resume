compositeSvc
service in module RA.IPDisplay

Description

Provides functionality specific for rendering displays with multiple cells

Methods

  • getMobileHeightOfCell(originalWidth, originalHeight, isXs)

    Returns the height of the specified cell on mobile viewport.

    Parameters
    ParamTypeDetails
    originalWidthNumber

    Width (number of columns) of cell in desktop display.

    originalHeightNumber

    Height (number of rows) of cell in desktop display.

    isXsBoolean

    Flag which indicates if we are on larger mobile device or on extra small.

    Returns
    Number

    Number of rows the cell should cover on mobile viewport.

  • isMobile()

    Returns whether size of viewport is mobile sized. It is comparing only the width of viewport. Uses raScreenSizeSvc from mobile-toolkit-ra and its "sm" and "xs" constants.

    Returns
    Boolean

    Indication whether the width of display is mobile sized. True if width of the viewport is under "sm" constant from raScreenSizeSvc.

  • isStyleObjectValid(style)

    Checks whether given style object contains dashboardBackgroundColor, cellBackgroundColor and displayPalette properties. If displayPalette property is not set, it assigns default palette using "getDefaultPalette" function

    Parameters
    ParamTypeDetails
    styleObject

    Object representing the style of display.

    Returns
    Boolean

    False if object itself, or its properties dashboardBackgroundColor or cellBackgroundColor are not defined.

    Otherwise it returns true.

  • isXsSize()

    Returns whether size of viewport is extra small. It is comparing only the width of viewport. Uses raScreenSizeSvc from mobile-toolkit-ra and its "xs" constant.

    Returns
    Boolean

    Indication whether the width of display is "extra small". True if width of the viewport is under "xs" constant from raScreenSizeSvc.

  • layoutSort(array)

    Sorts the input array by the position of cells (at first by row and then by column).

    Example of usage can be found in raIpComposite directive documentation.
    Parameters
    ParamTypeDetails
    arrayArray

    Array of cells which should be sorted. Each object in array has to have x and y number properties.
