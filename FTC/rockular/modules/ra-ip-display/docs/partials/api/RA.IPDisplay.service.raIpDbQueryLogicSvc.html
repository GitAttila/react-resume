raIpDbQueryLogicSvc
service in module RA.IPDisplay

Description

Basically this service is the functional part of raIpdbQuery directive. Scope passed around is the scope which directive is creating. Besides the bound variables (like rbParameters, riFqn and so on) scope holds up the information which is passed into or related to ra-data-grid object:

  • scope.fqn is an fqn for query.

Properties

  • widgetStates

    Widget state object with set of fixed values of following value:

    • NODATA - it means widget has no data
    • LOADING - it means that the widget and the data are currently loaded
    • REFRESHING - it defines that chart is refreshed
    • READY - it means that chart has already loaded the data
