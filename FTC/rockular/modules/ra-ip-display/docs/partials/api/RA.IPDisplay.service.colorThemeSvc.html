colorThemeSvc
service in module RA.IPDisplay

Description

Provides functionality to work with color themes and palettes

Palettes consist of following:

  • White and Black color
  • 8 palette specific colors
  • 6 shades from each basic color -> 60 (6 * (8 + 2)) color total

  • getNewThemePalette(newPalette, oldColor)

    Returns color from newPalette on the position which is specified in oldColor.index. If newPalette does not exist, it returns the color from default palette. If color does not have specified index, or the index is invalid, it throws RAError.

    If the color is in string format it is returned as it is.

    Parameters
    ParamTypeDetails
    newPaletteString

    Name of new palette

    oldColorObjectString

    Color object (see colorTypeSvc) or hexadecimal String describing the old color

    Returns
    Object

    If the color is THEME_COLOR it returns color object (see colorTypeSvc) describing color in new palette on same index as the oldColor was in old palette. Else it returns unchanged oldColor parameter.
