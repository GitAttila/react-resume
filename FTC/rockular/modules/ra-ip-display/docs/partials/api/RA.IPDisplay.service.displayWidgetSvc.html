  • decodeValue(rootFqn, value)
    

    Returns object with information about the value input.
    

    Parameters
    ParamTypeDetails
    rootFqnString
    String representing the anchor FQN. Should be without fqnPrefix.
    

    valueString
    String representing the value to be decoded.
    

    Returns
    Object
    Object with two properties.
    

      

    • isConstant: Boolean value which is true when value parameter does not contain fqnPrefix.
      • 

    • value: String property.
        

      • If isConstant is true, it returns unmodified value input parameter.
        • 

      • If isConstant is false, it returns absolute FQN without fqnPrefix. This is built based on
rootFqn parameter which the value is relative to. If the original value parameter was not relative
(do not start with delimiter character) the rootFqn parameter is not used.
        • 

      

      • 

    

    If the value is relative FQN and rootFqn is not specified, it throws RAError.
    

    Example

  • getFullFqnValue(rootFqn, value)
    

    Get full FQN out of rootFqn and value.
This method decodes value and returns proper full FQN or undefined in case of invalid FQN.
This method internally uses RA.IPDisplay.service:displayWidgetSvc.decodeValue.
This method throws error in case there is rootFqn invalid (undefined, null, false) or empty string.
    

    Parameters
    ParamTypeDetails
    rootFqnString
    the root of the fqn
    

    valueString
    the value to decode fqn from
    

    Returns
    String|undefined
    Full fqn without prefix or undefined in case of constant value.
