momentHelperSvc
service in module RA.IPDisplay

Description

Service which provides functions which help with moment library and datetime.

Methods

  • createUTCMomentFromString(dateString)

    Helper method to create moment date from supported datetime string formats. Supported formats:

    • all ISO_8601, long and short parts (up to "YYYY-MM-DDTHH:mm:ss.SSSSZ" and "YYYYMMDDTHH:mm:ss.SSSSZ")
    • RFC822 - Mon, 15 Aug 05 15:52:01 +0000
    • RFC2822 - Mon, 15 Aug 2005 15:52:01 +0000
    • "YYYY-MM-DD" up to "YYYY-MM-DD HH:mm:ss.SSSSZ"
    • other custom and non-standard formats:
      • en_US "YYYY-MM-DD" up to "MM/DD/YYYY HH:mm:ss:SSSSZ"
      • fr-FR "DD/MM/YYYY" up to "DD/MM/YYYY HH:mm:ss:SSSSZ"
      • ja-JP "YYYY/MM/DD" up to "YYYY/MM/DD HH:mm:ss.SSSSZ"
      • javascript Date to string format "ddd MMM DD YYYY HH:mm:ss [GMT]Z"
    Parameters
    ParamTypeDetails
    dateStringString

    string to create moment from

    Returns
    moment

    moment date, check if moment is valid later
