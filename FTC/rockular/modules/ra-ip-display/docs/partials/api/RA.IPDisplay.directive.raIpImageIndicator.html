raIpImageIndicator
directive in module RA.IPDisplay

Description

The raIpTrend directive is responsible for rendering the Image Indicator Widget.

Usage

as element:
<ra-ip-image-indicator
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}">
</ra-ip-image-indicator>

Directive info

  • This directive creates new scope.
