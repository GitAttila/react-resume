raIpVerticalGauge
directive in module RA.IPDisplay

Description

The raIpVerticalGauge directive is responsible for rendering the Linear gauge widget in vertical orientation.

Usage

as element:
<ra-ip-vertical-gauge
       rb-paremeters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]>
</ra-ip-vertical-gauge>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParemetersobject

An object representing the parameters of Vertical Gauge. This object contains array of objects (rbParameters.items) with these properties:

Property Type Details Default value
value Number The main property, number value of gauge
caption (optional) String The text which is displayed as caption of gauge This property is autobound to model .Name property of tag or it is empty
unit (optional) String The text which is displayed as part of caption of gauge in []. This property is autobound to model .EngineeringUnit property of tag or it is empty
min (optional) Number The minimum value on scale. This property is autobound to model .Min property of tag or it is empty
max (optional) Number The maximum value on scale. This property is autobound to model .Max property of tag or it is empty
precision (optional) Number The decimal precision of value Default value is 2, minimum value is 0, maximum value is 4
defaultColor (optional) Object The default color of gauge bar. Color object(see * colorTypeSvc) or hexadecimal String for compatibily reasons #00a2e8
defaultFontColor (optional) Object The default color of value box etx. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #7f7f7f
outOfRangeColor (optional) Object The color of gauge bar when value is outside Min and Max. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #e22028
outOfRangeFontColor (optional) Object The color of value box text when value is outside Min and Max. Color object see * colorTypeSvc) or hexadecimal String for compatibily reasons #e22028
ranges (optional) Array The array with ranges for gauge
ranges.to (optional) Number The beggining of range
ranges.from (optional) Number The end of range
ranges.color (optional) String The color of gauge bar when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #00a2e8
ranges.fontColor (optional) String The color of value box text when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #7f7f7f
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.
