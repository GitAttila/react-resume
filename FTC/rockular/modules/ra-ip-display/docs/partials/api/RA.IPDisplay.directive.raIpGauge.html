raIpGauge
directive in module RA.IPDisplay

Description

The raIpGauge directive is responsible for rendering the Gauge widget.

Usage

as element:
<ra-ip-gauge
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]
       [rb-custom="{object}"]>
</ra-ip-gauge>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of Gauge. This object contains array of objects (rbParameters.items) with these properties.

Property Type Details Default value
value Number The main property, value of gauge
caption (optional) String The text which is displayed as caption of gauge This property is autobound to model .Name property of tag or it is empty
subcaptionSwitch (optional) Boolean Flag if subcaption is used false
subcaption (optional) String The text which is displayed as subcaption of gauge This property is autobound to model .Name property of tag or it is empty
unit (optional) String The text which is displayed as part of caption of gauge in []. This property is autobound to model .EngineeringUnit property of tag or it is empty
min (optional) Number The minimum value on scale. This property is autobound to model .Min property of tag or it is empty
max (optional) Number The maximum value on scale. This property is autobound to model .Max property of tag or it is empty
precision (optional) Number The decimal precision of value Default value is 2, minimum value is 0, maximum value is 4
multiplication (optional) Number The multiplication factor for value 1
tickMarks (optional) Number The count of ticks around the scale Deafult value is 21, minimum is 3, maximum 101
numberMarks (optional) Array Step on which numer should show up on scale Deafult is 10, Possible values: 0, 1, 2, 5, 10
setPoint (optional) Number Value on which setpoint should be set
normalOperatingRangeSwitch (optional) Boolean Flag if normal operating range is used false
normalOperatingRangeStart (optional) Number The beggining of Normal Operarting Range
normalOperatingRangeEnd (optional) Number The end of Normal Operarting Range
disableGrayscaleRanges (optional) Boolean Flag if grayscale color should be disabled for ranges false
disableEmptyRanges (optional) Boolean Flag if transparent ranges visual should be disabled false
ranges (optional) Array The array with ranges for gauge
ranges.to (optional) Number The beggining of range
ranges.from (optional) Number The end of range
ranges.threshold (optional) Boolean Flag if range should be displayed as threshold false
ranges.color (optional) String The color when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #22b14c
ranges.fontColor (optional) String The color of text when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #ffffff
ranges.rangeIcon (optional) String The alarm icon for box when value is inside range. Possible values: none, low, medium, high, urgent none
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.

rbCustom
(optional)
object

This parameter is not currently not used.
