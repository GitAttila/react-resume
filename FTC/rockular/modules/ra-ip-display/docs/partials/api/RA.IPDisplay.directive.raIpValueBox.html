raIpValueBox
directive in module RA.IPDisplay

Description

The raIpValueBox directive is responsible for rendering the valuebox widget.

Usage

as element:
<ra-ip-value-box
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}">
</ra-ip-value-box>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of ValueBox. This object contains array of objects (rbParameters.items) with these properties:

Common Parameters

Property Type Details Default value
maximumFontSize Number Maximum font size, it can be between 10 and 40 14

Schema Type: Number

Property Type Details Default value
value Number The main property, number value of valuebox
numberFormat (optional) Array String enum defining format of number (General, Numeric, Currecny, Percent) General
separator (optional) Boolean Only for numberFormat Numeric, defines if thousands separator is used false
prefix (optional) String Only for numberFormat Currency, prefix before number
suffix (optional) String Only for numberFormat Currency, suffix after number
caption (optional) String The text which is displayed as caption of valuebox This property is autobound to model .Name property of tag or it is empty
unit (optional) String The text which is displayed as part of caption of valuebox in []. This property is autobound to model .EngineeringUnit property of tag or it is empty
min (optional) Number The minimum value for outOfRange colors. This property is autobound to model .Min property of tag or it is empty
max (optional) Number The maximum value for outOfRange colors. This property is autobound to model .Max property of tag or it is empty
precision (optional) Number The decimal precision of value Default value is 2, minimum value is 0, maximum value is 4

outOfRangeFontColor (optional) Object The color of text when value is outside Min and Max. Color object see * colorTypeSvc) or hexadecimal String for compatibily reasons #ffffff
ranges (optional) Array The array with ranges for valuebox
ranges.from (optional) Number The beggining of range
ranges.to (optional) Number The end of range
ranges.color (optional) Object The color of box when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #22b14c
ranges.fontColor (optional) Object The color of text when value is inside range. Color object (see * colorTypeSvc) or hexadecimal String for compatibily reasons #ffffff
