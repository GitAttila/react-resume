raIpWidgetHeader
directive in module RA.IPDisplay

Description

Widget live mode controls, use to allow user interactively manage live mode properties of certain widget.

Usage

as element:
<ra-ip-widget-header
       config="{Object}">
</ra-ip-widget-header>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
configObject

An object with header bar configuration.

Property Type Details Default value
autoReload Object Object with properties isActive, period in seconds is autoreload interval
timeHorizonItems Array Array with object definitions for duration select box.
anchorTime Date End date to count start time for custom range. Start time is calculated as end time minus duration. null
duration Number Actual negative duration which will also be selected in selectbox defined by timeHorizon items.
showExportButton Boolean Flag indicating that widget header should show export button (before refresh button)
showBar Boolean Flag indicating that widget header should show as bar true
hideLiveMode Boolean Flag indicating that widget header have to hide live mode settings false
