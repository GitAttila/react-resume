raIpDbQuery
directive in module RA.IPDisplay

Description

This raIpDbQuery directive is responsible for rendering Query Table Widget.

Usage

as element:
<ra-ip-db-query
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]>
</ra-ip-db-query>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of Query Table Widget.

Query Table Widget Specific Parameters

Property Type Details Default value
dbQueryTitle (optional) String DB Query Title for Query Table This property is autobound to model .Name property of tag or it is empty
query String FQN of the query instance in data model
parameters Array query parameters array of TYPE-VALUE pairs, where Value is of given type
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.
