transformationSvc
service in module RA.IPDisplay

Description

Service which provides functions to transform outdated parameters of display.

Methods

    Function which adds layout information into the configuration (x, y, width and height properties for each cell).

    Parameters
    ParamTypeDetails
    paramsObject

    Display configuration.

  • registerTransformationHooks(hooks)

    Registers the hooks for transformation from version to version.

    Parameters
    ParamTypeDetails
    hooksObject

    Object with hook functions. E.g. 

        {
     updateDisplayConfigurationVersion1ToVersion2: function (config) {...},
     updateDisplayStyleUnversionedToVersion1: function (style) {...}
    }

  • transformConfiguration(params)

    Main function which adapts configuration from old format to a new one.

    Parameters
    ParamTypeDetails
    paramsObjectString

    Old configuration, params object is altered after method is run.

    Returns
    Object

    Configuration valid for current version of raIpComposite.
