raIpBarChart
directive in module RA.IPDisplay

Description

This raIpBarChart directive is responsible for rendering Bar Chart Widget.

Usage

as element:
<ra-ip-bar-chart
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]>
</ra-ip-bar-chart>
as attribute
<ANY ra-ip-bar-chart
     rb-parameters="{object}"
     ri-fqn="{string}"
     rb-is-active="{boolean}"
     ro-schema="{function}"
     ro-examples="{function}"
     [rb-style="{string}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of Bar Chart Widget.

Common Parameters

Property Type Details Default value
orientation String Orientation of the Chart (horizontal/vertical) vertical
dataMode (optional) String Data mode for what data is to be displayed in widget (liveMode/timeMode) liveMode
autosize (optional) Boolean Whether to automatically set the size of the wiget when dataMode is liveMode (true/false) true
timePeriod (optional) Number The time period in seconds for which data to be displayed when dataMode is timeMode 300
aggregate (optional) String Data Aggregation method for widget display when dataMode is timeMode (Average/Sum/Minimum/Maximum/Variance/StandardDeviation) Average
dataSampling (optional) Number Data Sampling Rate in seconds for when the dataMode is timeMode (-1 means that data sampling is off) -1

Bar Chart Widget Specific Parameters

Property Type Details Default value
caption (optional) String Caption of the Bar Chart This property is autobound to model .Name property of tag or it is empty
color (optional) String Color of the Bar Chart #26a9e0
max (optional) Number Maximum Value This property is autobound to model .Max property of tag or it is empty
min (optional) Number Minimum Value This property is autobound to model .Min property of tag or it is empty
precision (optional) Number Decimal Precision for the Value 2
unit (optional) Number Engineering Unit of the Value This property is autobound to model .EngineeringUnit property of tag or it is empty
value String FQN of the numeric item in data model undefined
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.
