raIpMap
directive in module RA.IPDisplay

Description

The raIpMap directive is responsible for rendering the Map Widget.

Usage

as element:
<ra-ip-map
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]
       ro-params-changed="{function}">
</ra-ip-map>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing common parameters and the parameters of Map Widget.

Common Parameters

Property Type Details Default value
center (optional) String The geographic coordinates of center of map 0,0
zoom (optional) Number The zoom level of map, minimum value is 0, maximum value is 18 0
hideLegend (optional) Boolean Disable legend in Map widget false
clusterSize (optional) Number Radius of cluster for markers, minimum vlaue is 10, maximum is 20 15
popupVerbosity String The enum with options which minimum info display in cluster popup. Values are - disabled, low, medium, high, urgent urgent
indication (optional) Array The array with members of legend for Map widget
indication.from (optional) Number The beginning of indication range
indication.to (optional) Number The end of indication range
indication.color (optional) Object The color of legend member. Color object(see * colorTypeSvc) or hexadecimal String for compatibly reasons #ffffff
indication.label (optional) String The text of legend member
indication.verbosity (optional) String The enum with option which verbosity level range is. Values are - disabled, low, medium, high, urgent disabled

Map Widget Specific Parameters

Property Type Details Default value
value String Main property, geographic coordinates of marker
caption (optional) String The caption of markers popup
text (optional) String The text inside of markers popup
colorIndication (optional) Number The value which determines color of marker based on legend
drillThrough (optional) String The enum with options to add link to markers popup. Values are - disabled, displayRel, url disabled
drillLink Object or String For drillThrough set to displayRel this is link to another display. For drillThrough set to url this is external url link
drillCaption String Valid only for drillThrough set to displayRel or url. Caption of link.
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget should be active.

roSchemafunction

Function which should be called to initialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to initialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.

roParamsChangedfunction

Function which should be called when center of map was changed by mouse event
