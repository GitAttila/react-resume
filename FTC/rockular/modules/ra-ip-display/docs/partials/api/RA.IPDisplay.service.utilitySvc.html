utilitySvc
service in module RA.IPDisplay

Description

Provides basic utilities.

Methods

  • camelCase(name)

    Returns camelCase name, converted name to camelCase form, based on special characters. Based on code from Angular 1 https://github.com/angular/angular.js/pull/15249/files and https://github.com/angular/angular.js/blob/master/src/ng/compile.js#L3605 because $normalize is not public.

    Parameters
    ParamTypeDetails
    namestring
    • name with special symbols to be converted, symbols '-', ':', '_' will be replaced and next character converted to uppercase
    Returns
    string

    camelCase string name.

  • isMobile()

    Checks if user browser/navigator is mobile one based on window navigator userAgent string.

    Returns
    boolean

    true when user is using mobile browser

  • normalizeDirectiveName(name)

    Returns normalized directive name. Converts name (e.g. dash/colon/underscore-delimited string, optionally prefixed with x- or data-) to its normalized, camelCase form.

    Parameters
    ParamTypeDetails
    namestring
    • name to be converted
    Returns
    string

    Normalized string name.

    Example

    The example below shows usage of normalizeDirectiveName method and result.

    • data-ra-example: Result 'raExample'
    • x-ra-example: Result 'raExample'
    • example: Result 'example'
    • raExample: Result 'raExample'
