widgetsConfigurationSvc
service in module RA.IPDisplay

Description

Service providing information about widgets in current implementation.

Methods

  • getComposerComponents()

    Returns
    Array

    The list of widgets which was set during initialization.

  • getComposerDefaults()

    Returns
    Object

    The list of default widgets bound to specified types which was set during initialization.

  • init(_composerComponents, _composerDefaults)

    Initializes the list of accessible widgets. If _composerComponents is specified, it uses this array. Otherwise it use getWidgetsConfiguration operation on Widget Library item in the model on Widget library fqn to obtain this list.

    Parameters
    ParamTypeDetails
    _composerComponentsArray

    Array containing accessible widgets. Each object in array should contain following properties:

    • directive: Name of widget directive (in HTML valid format, e.g. data-ra-example)
    • name: Label which will be used for widget
    • category: Optional. Reference to existing widget category (curently implemented only in Composer which si not a part of this module).
    _composerDefaultsObject

    Object which defines what widget should be used as default one for specified types. Only used in Composer which is not a part of this module. Example: 

     {
   "number": "data-ra-ip-gauge",
   "boolean": "data-ra-ip-indicator",
   "string": "data-ra-ip-value-box",
   "mimeImagePng": "data-ra-ip-image-container",
   "mimeImageJpg": "data-ra-ip-image-container",
   "mimeImageGif": "data-ra-ip-image-container",
   "mimeImageSvgXml": "data-ra-ip-image-container"
  }

  • isReady()

    Returns
    Boolean

    Indication whether the list of accessible widgets is already initialized

  • isValidWidgetType(type)

    Checks if given type is specified in composerComponents array.

    Parameters
    ParamTypeDetails
    typeString

    Directive name which should Be checked.

    Returns
    Boolean

    true if type is undefined or is in the composerComponents array. Otherwise returns false.
