raIpImageContainer
directive in module RA.IPDisplay

Description

The raIpImageContainer directive is responsible for rendering the Raster Image Widget.

Usage

as element:
<ra-ip-image-container
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}">
</ra-ip-image-container>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing common parameters and the parameters of Image Widget.

Image Widget Specific Parameters

Property Type Details Default value
source String Source of the Image (fromModel/noReb/URL) fromModel
fqn (optional) String If the Source is fromModel, FQN of the Image in the data model undefined
URL (optional) String If the Source is URL, URL string of the image to display undefined
horizontalAlign (optional) String Horizontal Alignment of Image (left/right/center) center
keepRatio (optional) Boolean Keep the image ratio in proportion true
autoSize (optional) Boolean Autosize the image container as per image size false
verticalAlign (optional) String Vertical Alignment of Image (top/bottom/center) center
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.
