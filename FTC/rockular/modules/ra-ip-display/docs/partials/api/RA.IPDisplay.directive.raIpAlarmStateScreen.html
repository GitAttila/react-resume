raIpAlarmStateScreen
directive in module RA.IPDisplay

Description

The raIpAlarmStateScreen directive is responsible for rendering the AlarmState Screen Widget.

Usage

as element:
<ra-ip-alarm-state-screen
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]
       rb-design-mode="{boolean}">
</ra-ip-alarm-state-screen>

Directive info

  • This directive creates new scope.
