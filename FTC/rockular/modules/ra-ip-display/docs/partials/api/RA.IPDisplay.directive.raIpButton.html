raIpButton
directive in module RA.IPDisplay

Description

The raIpButton directive is responsible for rendering the Button Widget.

Usage

as element:
<ra-ip-button
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]
       [rb-in-design-mode="{boolean}"]>
</ra-ip-button>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing common parameters and the parameters of Button Widget.

Common Parameters

Property Type Details Default value
size (optional) String Size of the Button to Draw (xsmall/small/medium/large) medium

Button Widget Specific Parameters

Property Type Details Default value
drillThrough String Drill Through object to be opened when button is clicked (disabled\displayRel\display\documentRel\document\url) disabled
drillLink (optional) String When drillThrough is not disabled, URL or FQN ot item to navigate to when button is clicked undefined
drillCaption (optional) String When drillThrough is not disabled, the caption of the button This property is autobound to model .Name property of Link item or it is empty
style (optional) String Style of the button (none\default\primary\success\info\warning\danger) default
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.

rbInDesignMode
(optional)
boolean

True if the outer environment is in design mode.
