raIpTrend
directive in module RA.IPDisplay

Description

The raIpTrend directive is responsible for rendering the Trend Widget.

Usage

as element:
<ra-ip-trend
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}"
       [rb-style="{string}"]
       rb-design-mode="{boolean}">
</ra-ip-trend>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing common parameters and the parameters of Trend Widget.

Common Parameters

Property Type Details Default value
autoReload Boolean Should the values be auto reload False
autoReloadInterval (optional) Number Time interval to auto reload after 10 sec
chartTitle (optional) String Title of the widget "Title"
lastSeconds (optional) Number Number of Seconds the value are being displayed for 300 sec
highlightTrace (optional) Number Number of times active range should be bigger, values: 1,2,3,4 1
minMax (optional) Number The style min/max values to be displayed with (using RA_CHART_CONST.extremeLevels constants) 2 - TagValues
hideYAxis (optional) Boolean Hide Y-axis in Trend false
hideInterpolatedQualityValues (optional) Boolean Hide interpolated quality values in traces false
showBadQualityValues (optional) Boolean Show bad quality values in traces false
hideTimeControl (optional) Boolean Hide timecontrol for Trend in runtime false
captionSettings (optional) String Setting for caption format ("" - last part of FQN, auto - automatic format, fqn - full FQN) ""
showSpikes (optional) String Setting to display spike values on Trend ("" - no spikes, MaximumActualTime - display maximum values, MinimumActualTime - display minimum values ""

Trend Widget Specific Parameters

Property Type Details Default value
caption (optional) String The text which is displayed as caption of trace This property is autobound to model .Name property of tag or it is empty
unit (optional) String The text which is displayed as part of caption of trend in []. This property is autobound to model .EngineeringUnit property of tag or it is empty
precision (optional) Number The decimal precision of value Default value is 2, minimum value is 0, maximum value is 4
min (optional) Number The minimum value on scale. This property is autobound to model .Min property of tag or it is empty
max (optional) Number The maximum value on scale. This property is autobound to model .Max property of tag or it is empty
color (optional) Object Default color trend would be displayed in #26a9e0
readAxisRange Boolean Read the Range of Axis false
useMinMaxYaxis Boolean Use Min Max for Y-axis false
tail String FQN of the tag to be displayed in Trend
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.

rbStyle
(optional)
string

String indicating which class should be used in widget.

rbDesignModeboolean

Flag which indicates whether the widget is in design mode.
