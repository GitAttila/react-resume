styleSvc
service in module RA.IPDisplay

Description

Service responsible for adding custom css rules to the appended stylesheet.

Methods

  • applyStyles(style)

    Adds the custom CSS rule to DOM. It changes defintion of background-color property for .ra-dashboardStyleCustom-dashboard-background and .ra-dashboardStyleCustom-cell-background selectors.

    Parameters
    ParamTypeDetails
    styleObject

    Object containing cellBackgroundColor and dashboardBackgroundColor properties. These should contain any valid CSS color definition.
