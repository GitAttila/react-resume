raIpLabel
directive in module RA.IPDisplay

Description

The raIpLabel directive is responsible for rendering the Label widget for visualizing simple texts.

Usage

as element:
<ra-ip-label
       rb-parameters="{object}"
       ri-fqn="{string}"
       rb-is-active="{boolean}"
       ro-schema="{function}"
       ro-examples="{function}">
</ra-ip-label>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rbParametersobject

An object representing the parameters of Indicator. This object contains array of objects (rbParameters.items) with these properties. This This widget supports only 1 item in rbParameters.items array.

Schema type: String

Property Type Details Default value
value String The main property, string value or FQN for live data
font (optional) String Font to be used in label. Value should be from following enumeration: robotoFont, serifFont, sansserifFont, monospaceFont robotoFont
fontColor (optional) Object Color for label. Color object(see colorTypeSvc) or hexadecimal String for compatibily reasons #000000
fontBold (optional) Boolean If true, label font is bold false
fontItalic (optional) Boolean If true, label font is italic false
textUnderline (optional) Boolean If true, label is underlined false
fontSize (optional) Number Size of the font 20
horizontalAlign (optional) String Horizontal alignment of label. Value should be from following enumeration: left, right, center, justify center
verticalAlign (optional) String Vertical alignment of label. Value should be from following enumeration: top, middle, bottom middle
riFqnstring

FQN the widget is relative to.

rbIsActiveboolean

Flag which shows whether the widget sould be active.

roSchemafunction

Function which should be called to intialize the widget schema in outside environment.

roExamplesfunction

Function which should be called to intialize the widget examples in outside environment.
