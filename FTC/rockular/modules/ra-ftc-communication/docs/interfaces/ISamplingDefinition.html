Properties

deltaT
deltaT: number
Type : number
Optional

If the slice count is zero, then a non-zero value representing the width of a time slice in seconds must be provided. A positive value means the slices are anchored at the start time of the requested time period. A negative value indicates anchoring at the end of the time period.

samplingType
samplingType: SamplingTypes
Type : SamplingTypes

One of the sampling types supported by the timeseries. See SamplingTypes enum.

sliceCount
sliceCount: number
Type : number
Optional

The number of the slices that the provided time period should be cut into. If zero then a non-zero DeltaT value needs to be provided.
