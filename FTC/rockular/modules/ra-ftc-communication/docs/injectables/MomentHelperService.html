Static applyTimezone
applyTimezone(datetime: string, timezone: string)

Method to apply timezone to UTC string. Returns date string in maximum javascript resolution possible, supported by using moment library (Javascript date - milliseconds). Resolution for time after milliseconds will be lost.

Parameters :
Name Type Optional Description
datetime string No

datetime string in UTC format
timezone string No

timezone to apply If working with timezone in datetime string it is recommended to use moment.parseZone to preserve timezone information.
Returns : any
