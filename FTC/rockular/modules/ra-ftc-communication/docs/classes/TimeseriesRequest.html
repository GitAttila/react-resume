Constructor

constructor(fqns: identifier[], timePeriod: ITimePeriod, samplingDefinition?: ISamplingDefinition, timeDeadBand?: string, valueDeadBand?: number, qualityFilter?: TimeseriesRequestQuality | number)

Timeseries request constructor

Parameters :
Name Type Optional Description
fqns identifier[] No

List of fqns (identificators) to be used for request
timePeriod ITimePeriod No

Settings for time period of timeseries request, e.g. start time, end time and duration
samplingDefinition ISamplingDefinition Yes

(Optional) definition for timeseries to set how to process data, which Sampling type to use
timeDeadBand string Yes

(Optional) Settings for time deadband - possible structure [ws][-] d | d.hh:mm[:ss[.ff]] | hh:mm[:ss[.ff]] [ws]
valueDeadBand number Yes

(Optional) Specifies the value dead band, can be any float number.
qualityFilter TimeseriesRequestQuality | number Yes

(Optional) This is a bit flag value with the following options: IncludeGood : 1 IncludeBad : 2 IncludeUncertain : 4 IncludeAll : 7 Use a value from predefined enum TimeseriesRequestQuality or specify a vallue.
