raIpCard
directive in module RA.IPComponents

Description

This component is responsible for rendering card. It allows to put inside another component. Card is only wrapper with hover effect which can be disabled.

Usage

as element:
<ra-ip-card
       config="{Object}">
</ra-ip-card>

Parameters

ParamTypeDetails
configObject
  • class {string} to add specific class
  • disabledHover {boolean} to disable hover effect
