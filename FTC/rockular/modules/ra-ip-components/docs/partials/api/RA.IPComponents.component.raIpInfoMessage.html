raIpInfoMessage
directive in module RA.IPComponents

Description

This component is responsible for rendering no data message with action button. The component expects to receive {object} config.

Usage

as element:
<ra-ip-info-message
       config="{object}">
</ra-ip-info-message>

Parameters

ParamTypeDetails
configobject
  • class {string}
  • text {string} text message
  • action {object} raIpActions component. See also RA.IPComponents.component:raIpActions.
