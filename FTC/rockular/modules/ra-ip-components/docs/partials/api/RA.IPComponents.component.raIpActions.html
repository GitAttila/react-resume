raIpActions
directive in module RA.IPComponents

Description

This component is responsible for rendering action buttons. Action button can be an icon or a button. Component allows to set size of each action and align for whole component. For icon button is used raIpBadge component and for button is used MFT button.

Usage

as element:
<ra-ip-actions
       config="{object}">
</ra-ip-actions>

Parameters

ParamTypeDetails
configobject
  • class {string} to customize style of component
  • align {string} left | center | right

  • actions {array} actions

    • class {string} class for icon button
    • disabled {boolean} disabled action button
    • onAction {function} called on click
    • tooltip {string} information text shown on hover on button
    • value {string} for button when this value is undefined it set icon button

    • size {string} small | medium | large

      raIpBadge component for icon action See also RA.IPComponents.component:raIpBadge.
