    Get started with InfoPlatform

    How to get started

    When using a NuGet repository you can either work with the NuGet client directly or with Visual Studio.

    Visual Studio Configuration

    To add the InfoPlatform repository as a Package Source under NuGet Package Manager:

    1. Access the corresponding repositories in the "Options" window (Options | Tools) and select to add another Package Source. Name: ENTER_RESOURCE_NAME (e.g. InfoPlatform NuGet repository)
    2. Paste the snippet below in the URL field

      https://cat.ra-int.com/artifactory/api/nuget/infoplatform-build

    3. Make sure it is checked.
    NuGet Client Configuration

    To add this repository to the list of sources of your NuGet client use the following command

    nuget sources Add -Name InfoPlatform -Source

    https://cat.ra-int.com/artifactory/api/nuget/infoplatform-build

    Adding the source to nuget.config

    You can also add the repository in either the global nuget.config or the nuget.config of your project by adding the following line to the 'packageSources' section:

    <add key="InfoPlatform" value="https://cat.ra-int.com/artifactory/api/nuget/infoplatform-build" />
