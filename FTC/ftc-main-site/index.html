about

Built on Microsoft Azure Platform as a Service (PaaS) and leveraging code and functionality from InfoPlatform, MFT and CSM, the FactoryTalk Cloud Platform is designed to provide a flexible base platform which specific applications can be built on top of.

The FactoryTalk Cloud Platform offers a secure and extensible framework for business units to build new applications. A variety of extensibility toolkits are available, making it possible for business units to create their own functionality across all levels of the FactoryTalk Cloud Platform.
